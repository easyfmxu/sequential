About

yolpo is a side project and mash-up development by lorenzo puccetti (https://github.com/lorenzoongithub ).

It was started over a few weekends and several late evenings some time during spring 2015, by combining various technologies I was having fun to play with.

Amongst its notable parts:

  • On the Server Google App Engine
  • JS Parser - Esprima.org
  • HTML - sessionStorage, iframe
  • 3rd party services Gist GitHub API
  • Minifiers - YUICompressor, kangax html-minifier
