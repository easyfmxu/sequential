Basic Concepts

What is yolpo ?

yolpo is both a web site (http://www.yolpo.com) and a tool to visualize the execution of javascript.

What is yolpo for ?

It is for javascript API authors to present their work. But it can also be used as a regression test tool. yolpo helps you to share use cases against your javascript API, so that you can focus on your solution rather than worrying about how to present it.
