Not another code playground

On the surface yolpo might look like another code playground (jsfiddle, codepen, jsbin, ...)

Like a code playground it encourages users to try things out, provides the ability of editing code on the browser, and exploits Iframe to sandbox environments; but that's where the similarities end.

yolpo is not executing code to create a page, is creating a page to show code execution.
