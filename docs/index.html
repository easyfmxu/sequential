How does it work ?

At its core, sequential is a statement-by-statement JavaScript interpreter. This means that it does not eval code in its entirety. Instead, it breaks the code in sequential statements and starts by executing the first statement and continue by executing all statements in sequence.

This is a very natural way of presenting a program, a great fit for error discovery and simplifies comparisons.

The Environment

  • When running your script sequential gives you a new global function load(stringURL) to let you load any external JavaScript library.
  • Use console.log( ... ) to display any valuable output from your code
